This theme is built on Poole - The Jekyll Butler. Poole provides everything a developer needs to quick start a Jekyll project.

It is built by the great @mdo himself. I admire his work on Bootstrap and love pretty much what he does. So here it is, Fission: An ultra-lite Jekyll theme.
